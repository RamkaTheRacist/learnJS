!!!alert!!!
кнопкa «ОК».
Модальное окно с сообщением и кнопка «OK».
/////////////////
!!!prompt!!!
result = prompt(title(Текст для отображения в окне), [default]); 
Этот код отобразит модальное окно с текстом, полем для ввода текста и кнопками OK/Отмена.
Необязательный второй параметр, который устанавливает начальное значение в поле для текста в окне, которое видит юзер.
Пользователь может напечатать что-либо в поле ввода и нажать OK. Введённый текст будет присвоен переменной result. 
Пользователь также может отменить ввод нажатием на кнопку «Отмена» или нажав на клавишу Esc. В этом случае значением result станет null.
(!)Второй параметр является необязательным, но если не указать его, то Internet Explorer вставит строку "undefined" в поле для ввода.
////////////////
!!!confirm!!!
result = confirm(question);
Функция confirm отображает модальное окно с текстом вопроса question и двумя кнопками: OK(return true) и Отмена.
///////////////
Модальный метод: останавливают выполнение скриптов и не позволяют пользователю взаимодействовать с остальной частью страницы до тех пор, пока окно не будет закрыто.
На все указанные методы распространяются два ограничения:
1)Расположение окон определяется браузером. Обычно окна находятся в центре.
2)Визуальное отображение окон зависит от браузера, и мы не можем изменить их вид.
